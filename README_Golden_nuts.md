
# Dry Fruits Products Dataset Project

## Introduction
This project focuses on creating a structured dataset of dry fruit products commonly sold on e-commerce platforms. The dataset is intended for academic assignments, data analysis, and machine learning practice.

## Objective
To generate a clean and realistic dry fruits dataset suitable for analytical and predictive tasks.

## Dataset Description
- Total Products: 300
- File Format: CSV
- Domain: Dry Fruits

### Columns
1. Product_ID – Unique identifier
2. Product_Name – Name of the dry fruit product
3. Brand – Brand or supplier
4. Dry_Fruit_Type – Type of dry fruit
5. Pack_Size – Packaging size
6. Price_INR – Price in Indian Rupees
7. Rating – Customer rating
8. Number_of_Reviews – Review count

## Tools & Technologies
- Python
- Pandas
- Jupyter Notebook

## Workflow
1. Identify dry fruit attributes
2. Generate realistic values
3. Store in CSV format
4. Use for analysis

## How to Run
1. Open the notebook
2. Load CSV using pandas
3. Perform analysis or visualization

## Conclusion
This dataset provides a practical and realistic representation of dry fruit products for academic use.
